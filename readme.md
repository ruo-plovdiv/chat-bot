# Ръководство за инсталиране на Ubuntu с Rufus

Следвай стъпките по-долу, за да създадеш бутващ USB диск и да инсталираш Ubuntu на компютъра си.

## 1. Изтегляне на Rufus
- Изтегли [Rufus](https://rufus.ie/), който ще използваме за създаване на бутващия USB диск.

## 2. Изтегляне на Ubuntu
- Отиди на официалния сайт на Ubuntu: [https://ubuntu.com/download/desktop](https://ubuntu.com/download/desktop).
- Избери версия с **LTS (Long Term Support)**. Това е версия с дългосрочна поддръжка и е препоръчителна за начинаещи.
- Изтегли ISO файла на Ubuntu.

## 3. Подготовка на USB устройството
1. Включи USB устройството в компютъра си.
2. Стартирай **Rufus**.
3. В **Rufus**, избери:
   - **USB устройството**, на което ще инсталираш Ubuntu.
   - Избери ISO файла на Ubuntu, който си изтеглил.
4. Избери следните опции:
   - **Partition Scheme**: GPT (за нови компютри с UEFI).
   - **File System**: FAT32 (по подразбиране).
   - **Cluster Size**: 4096 bytes.
5. Натисни **Start**, за да започне създаването на бутващия USB диск.
   - Внимание: Всички данни на USB устройството ще бъдат изтрити, така че се увери, че нямаш важни файлове на него.

## 4. Стартиране на компютъра от USB
1. След като Rufus завърши процеса, рестартирай компютъра си.
2. Влез в **BIOS/UEFI менюто** на компютъра (обикновено с натискане на клавиш като F2, ESC или Del веднага след стартиране).
3. Променяй реда на зареждане (Boot Order), за да стартираш от USB устройството.
4. Запази промените и рестартирай компютъра. 
5. Ще се появи меню с инсталацията на Ubuntu. Избери дали да инсталираш Ubuntu или да го изпробваш без да правиш промени в системата (Live mode).

## 5. Инсталация на Ubuntu
1. След като заредиш Ubuntu, ще имаш възможност да избереш как да го инсталираш:
   - **Install Ubuntu** (Инсталация на Ubuntu):
     - Можеш да избираш да инсталираш Ubuntu вместо друга операционна система или да го инсталираш заедно с Windows.
   - **Try Ubuntu** (Опитай Ubuntu без инсталация):
     - Това ти позволява да изпробваш Ubuntu без да променяш системата.

2. Ако избираш инсталация, следвай инструкциите на екрана. Ако не си сигурен как да избираш дяловете, можеш да използваш опцията за **Erase disk and install Ubuntu**, но се увери, че имаш резервни копия на важните си файлове.
